# Server for CARP-OJ
- Judge worker: [carp_judge_worker](https://github.com/imxieyi/carp_judge_worker)

## Requirements
- JDK 1.8+
- MongoDB

## Build
```bash
gradle build
```

## Setup Database
```bash
gradle addData
```

## Run
```bash
gradle bootRun
```
